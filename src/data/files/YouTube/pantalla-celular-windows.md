# Cómo transmitir pantalla del celular a Windows


Paso a paso para transmitir la pantalla del celular al escitorio de windows.

En [mi canal de youtube](https://youtu.be/lT1rHovN86M) hay un video del paso a paso:

1. Bajar archivos:
- <a target="_blank" href="https://mega.nz/file/6lFBnAYY#YanFe094ssBA3T5IT9nbNajq-gxPes_me5ziGTyOT3k">platform-tools_r28.0.0-windows.zip</a>
- <a target="_blank" href="https://mega.nz/file/6tUWlKiT#PUTe5FHlUMcgo4d8qHNjqI7R480Zucar_f-XfxLyAjk">scrcpy-win64.rar</a>
2. Descomprimir en una carpeta
3. Conectar celular en modo solo carga  
4. Ejecutar
```csharp
adb devices
scrcpy
```
